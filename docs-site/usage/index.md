# Usage

This section covers how to install ThirdEra, create a world, and use the system as a GM or player.

## Installing the system and creating a world

1. Install or symlink the ThirdEra system into Foundry VTT’s `Data/systems/` folder as `thirdera/`. (If you install from a release or clone the repository, place or link it so Foundry sees `Data/systems/thirdera/`.)
2. In the **Setup** screen, create a new **World** and choose **ThirdEra** as the game system.
3. After the world is created, the system’s compendium packs (races, classes, feats, spells, etc.) are available in the **Compendium** sidebar.

Once the world is ready, you can create **Characters** and **NPCs** from the **Actors** sidebar and use the character sheet, NPC sheet, and compendiums as described in the following pages.
