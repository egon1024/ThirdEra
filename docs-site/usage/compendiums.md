# Compendiums

ThirdEra ships with compendium packs containing SRD content. They appear in Foundry’s **Compendium** sidebar when you have a ThirdEra world loaded.

## What’s included

| Pack | Content |
|------|--------|
| **Races** | SRD races (Dwarf, Elf, Gnome, Half-Elf, Half-Orc, Halfling, Human). |
| **Classes** | SRD base classes (Barbarian, Bard, Cleric, Druid, Fighter, Monk, Paladin, Ranger, Rogue, Sorcerer, Wizard) with level tables, class skills, and class features. |
| **Skills** | SRD skills with ability associations and flags. |
| **Feats** | SRD feats with prerequisites (required feats, BAB, ability scores). |
| **Weapons** | SRD weapons (simple, martial, exotic) with damage and properties. |
| **Armor** | SRD armor and shields. |
| **Equipment** | Adventuring gear and containers. |
| **Spells** | SRD spells for levels 0–9, with levels by class and domain where applicable. |
| **Schools** | Schools of magic (for spell school and subschool). |
| **Conditions** | SRD conditions (e.g. Blinded, Shaken) with optional mechanical effects. |
| **Creature Types** | SRD creature types for NPCs. |
| **Subtypes** | SRD subtypes for NPCs. |
| **Class Features** | Class feature items referenced by classes (e.g. Evasion, Sneak Attack). |

All content is SRD-only; no proprietary data is included.

## Using compendiums in play

- **Drag onto a character or NPC** - Open a compendium, then drag an item (race, class, feat, weapon, spell, condition, etc.) onto the actor’s sheet. The item is added to that actor. For actors, you can also drag an **Actor** document from a compendium into the scene or Actors sidebar to create a copy.
- **Browsing** - Use the compendium sidebar to search and open packs. Item sheets open when you click an item; you can read descriptions and then drag the item onto a sheet if needed.
- **Spell List** - The system provides a **Spell List** app (e.g. from the sidebar or a character’s Spells tab) to browse spells by class or domain and add them to characters.

Items dragged from compendiums are **copies** on the actor; changing or deleting the compendium entry does not change copies already on a character or NPC. You can edit the copy on the sheet as needed (e.g. customizing a weapon or spell).

## Extending with more content

The system is designed so that additional compendiums (e.g. third-party packs or your own world compendiums) can provide more races, classes, feats, or spells. Use the same item types and structure as the system’s packs so that sheets and level-up recognize them. Creating items in the world (Items directory) and dragging them onto actors works the same way as compendium items.
