makes you not want to use it, you are absolutely free to make that decision,

While this module is currently under development, I won't be accepting code
contributions, though I will certainly accept feedback from testers.  At this
time, I am not soliciting reports of missing features, or requests for
functionality.  

All of the above is likely to change as the module progresses.
